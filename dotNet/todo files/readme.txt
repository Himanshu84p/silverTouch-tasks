Steps of implementation

1) Run the script file to generate the Database.

2) Open any one project TodoApp named is CodeFirst approach and DbFirstTodo is DBFirst approach.both are projects of TodoList performed with different approaches DBFirst and CodeFirst respectively.

3) On Opening any of the folder there will be a solution file which contain sol file open it will redirect to visual studio code.

4) If EntityFramework is no installed then to install it right click on manage nuGet Packages.

5) Once Nuget Packages are opened click on browse to install EntityFramework.

6) Now you have to make changes in web config file by change or edit connectionString.   

7) Now you can run the project.

Follow same steps for other project too.
